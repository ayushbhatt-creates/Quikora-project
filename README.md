# 🛒 Quikora

Quikora is a Blinkit-inspired quick commerce web application built using **HTML, CSS, and Vanilla JavaScript**.

## 🚀 Live Demo

🔗 https://6a4350173352e6950c415805--illustrious-pastelito-05ebc6.netlify.app/

## ✨ Features

* Product browsing and search
* Dynamic cart management
* Local Storage support
* Checkout with UPI & Cash on Delivery
* Responsive user interface

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Local Storage API
